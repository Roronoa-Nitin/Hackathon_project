# Hackathon Project

 Overview
This is a Flask-based web application that analyzes user input and generates a score and category.

 Features
- User input submission
- Score generation
- Category detection
- Dashboard with analytics
- SQLite database storage

 Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS

 How to Run
1. Clone the repository
2. Install dependencies
3. Run app.py
4. Open in browser
